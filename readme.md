# Description:
## This is the app for managing your washing machine

### you can:
* create, read, update, delete washing machine
* create, read, update, delete washing mode
* start washing machine with selected mode

# Start app:
1. Create database
	* CREATE DATABASE  washing_machine
	* HOST: "localhost",
	* USER: "postgres",
	* PASSWORD: "RNvxabpMbD",
	* DB: "washing_machine",

2. Start backend
	* cd backend
	* npm i
	* npm run server

3. Start frontend
	* cd frontend
	* npm i 
	* npm start
